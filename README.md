# twitter_clone

A basic twitter clone built in rails.

User will be able to:

* create an account and sign up/sign in

* post tweets

* read, like and retweet tweets

* leave comments on tweets

* delete their own tweets

* upload profile pic 

----------------------------------------

Created with:

* Ruby 3.1.2

* Rails 7.0.3

* Database: Postgresql 

* Tailwind CSS


Gems:

* devise

* sidekiq

* cssbundling
